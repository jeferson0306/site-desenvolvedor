Meu nome é Jeferson Siqueira, sou desenvolvedor de websites. Desenvolvo sites conteudistas, informativos, blogs, cardápios, e muito mais! Todos eles com a responsividade ideal, se adaptando para melhorar a experiência do usuário e de seus clientes.

O que você
precisa saber

Existem vários tipos de projetos, e cada projeto possui uma forma e um tempo específico para ser desenvolvido.


  • Site informativo

    • Apenas conteúdo escrito, informações para serevem divulgadas para os clientes, como o Wikipedia


  • Blogs

    • Conteúdo com postagens, galeria de fotos e vídeos, podemos utilizar como exemplo, sites de jornais ou de viagens


  • Softwares

    • Um site com ferramentas utilizáveis, podendo ser utilizadas tanto pelo usuário quanto pelo cliente, como o site do Ifood ou da Magazine luiza.


Entenda melhor
sobre os produtos

Durante o desenvolvimento, temos que ajustar a visualização para cada uma das possíveis ferramentas que o usuário irá utilizar. Portanto, temos que adaptar para smartTV, desktop/notebook, tablet e celulares, cada um deles com uma experiência única, mas onde fica melhor cada elemento? É aqui que entra o papel do UX/UI Designer, estudando onde o seu cliente irá ter uma melhor experiência com aquela plataforma.
